# my-web-portfolio
This webpage is about my career planning and my skills

## 🔗 Live Demo
Dynamic Link: **[https://matiasdelgado10.github.io/my-web-portfolio/]**

## 🛠 Technologies & Development Standards
* **Structure & Semantics:** HTML5 (Structured with semantic tags like `<main>`, `<section>`, `<article>`, and `<fieldset>` for accessibility and SEO optimization).
* **Styling & Layouts:** CSS3 (Custom grid layouts, responsive tables, interactive form designs, and multi-device viewport optimization).
* **Interactivity & Logic:** Vanilla JavaScript (Asynchronous DOM manipulation, dynamic list generation, and clean user-state management).
* **Tools & Environment:** Visual Studio Code, Git/GitHub.

## 🚀 Key Features & Architectural Modules

### 1. Interactive Action Plan Module (JavaScript Engine)
* **Dynamic Target Tracking:** Powered by a clean DOM manipulation script (`script.js`) that allows users to append new academic or technical milestones to a live workflow list.
* **Input Validation & Sanity Checks:** Utilizes robust `.trim()` checks and alerting mechanisms to prevent empty input states, dynamically rendering newly added items with custom real-time styling.

### 2. Embedded Projects & Business Intelligence Showcases
* **Data Science & Churn Prediction Pipeline:** Features a comprehensive technical walkthrough of an end-to-end Machine Learning project aimed at predicting customer attrition. Highlights feature preprocessing (One-Hot Encoding, data scaling), Exploratory Data Analysis (EDA), and ensemble modeling performance metrics (Accuracy, Recall, F1-Score).
* **Enterprise ERP & SAP Modern Guide (2026 Edition):** A deep-dive knowledge base covering integrated modular structures (FI, CO, SD, MM, PP, WM, HR, ABAP), architectural migrations to SAP S/4HANA, and a data-driven Decision Matrix mapping specific business processes to recommended modules.

### 3. Responsive Data Analytics Matrix
* Contains an aligned, responsive comparison matrix utilizing multi-row and multi-column layouts (`rowspan`/`colspan`) to evaluate industry demands, entry barriers, and employment viability scores across Data Science and SAP Consultancy paths.

### 4. Enterprise-Ready Candidate Contact System
* A robust, styled interaction form designed with semantic `<fieldset>` divisions, `<legend>` captions, grouping dividers, and structured input controls (radio selections, dropdowns, checkboxes). It allows prospective employers and recruiters to filter contact reasons based on specialized career branches (SAP Consultancy, Data Analytics, Full-Stack Web Development).

## 📂 Project Structure & Pages
* `index.html` - Professional landing pad with focus areas and the JavaScript-driven target planning module.
* `hakkinda.html` - Academic path, career priorities, and foundational MIS definitions (ERP, Data Science definition lists).
* `galeri.html` - Integrated data matrix tables, SAP architectural documentation, and the customer churn analytics showcase.
